HOW TO USE THIS PANCAKE QUEUE!

1. Type in a name and hit "Enter" to add them to the queue (comments are optional).

2. Click on any row to toggle the "Done?" box, which helps you track which pancake has been made.

3. Double click on any row to remove the row from the queue.

NOTE: There is no cache for this web app. Refreshing the page will clear the queue!
